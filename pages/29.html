🏅 Day 29 - mixin

2024/08/01

作業說明：利用 mixin 搭配 include 的方法讓 CSS 換一個寫法，省去重複寫 @media 的問題。

CSS

                  .header{
                    width: 500px;
                  }
                  
                  @media(max-width: 768px){
                    .header{
                    width: 13px;
                    font-size: 20px;
                    }
                  }
                  
                  @media(max-width: 568px){
                    .header{
                    font-size: 8px;
                    }
                  }
                  
                  .header a{
                    background: black;
                  }
                  
                  @media(max-width: 568px){
                    .header a{
                    background: white;
                    }
                  }

SCSS

                  @mixin pad{
                    @media(max-width: 768px){
                    @content;
                    }
                  }
                  
                  @mixin mobile{
                    @media(max-width: 568px){
                    @content;
                    }
                  }
                  
                  .header{
                    width: 500px;
                    @include pad {
                      width: 13px;
                                      font-size: 20px;
                    }
                  
                    @include mobile {
                      font-size: 8px;
                    }
                  
                    a{
                      background: black;
                      @include mobile {
                        background: white;
                      }
                    }
                  }
