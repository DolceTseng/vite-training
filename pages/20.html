🏅 Day 20 - CSS 權重覆蓋

2024/07/19

作業說明：請回答以下問題
口訣：先看權重再看先後
ID > class > tag

1.請問 p 標籤內文字是什麼顏色

    HTML

  • 
                      <p class="blue red black">這是藍色ㄉ文字</p>

    • CSS

  •                       p {
                        color: white;
                      }

                      .black {
                        color: black;
                      }
                      
                      .blue {
                        color: blue;
                      }
                      
                      .red {
                        color: red;
                      }
Answer: 紅色
2.請問 p 標籤內文字是什麼顏色

    HTML

  • 
                      <p class="a p">這是藍色ㄉ文字</p>

    • CSS

  •                         p {
                          color: white;
                        }
                        
                        .p {
                          color: blue;
                        }
                        
                        a {
                          color: red;
                        }
Answer: 藍色
3.請問 p 標籤內文字是什麼顏色

    HTML

  •                         
                      <p class="p blue" id="blue">這是藍色ㄉ文字</p>

    • CSS

  •                         .p {
                          color: blue;
                        }
                        
                        .blue {
                          color: red;
                        }
                        
                        p.blue {
                          color: green;
                        }
                        
                        #blue {
                          color: white;
                        }
Answer: 白色
